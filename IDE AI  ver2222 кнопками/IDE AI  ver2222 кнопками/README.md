# 🚀 AI App Builder Constructor (Combined Platform)

## 🔥 Основные функции

### 🤖 1. ИИ Агент и Помощник
- **Интерактивный чат:** Генерация готового кода, создание новых файлов и редактирование существующих по запросу.
- **Предложения и быстрые ответы:** Заготовленные промпты для частых задач.
- **Генерация UI:** Автоматическое использование Tailwind CSS.

### 📁 2. Файловый менеджер и IDE
- **Визуальное дерево файлов:** Создание, удаление, навигация.
- **Интегрированный редактор:** Monaco Editor (как в VS Code) с подсветкой синтаксиса, номерами строк и умным редактированием.

### 👁️ 3. Предпросмотр в реальном времени (Live Preview)
- **Мгновенный рендеринг:** Отображение HTML/React кода без перезагрузки всей страницы.
- **Симуляторы устройств:** Превью для Desktop, iPhone и Android-телефонов.

### 📱 4. Мобильная сборка (Android / iOS via Capacitor)
- Управление и синхронизация Capacitor прямо из конструктора.
- Простая генерация `app-debug.apk` и синхронизация с Xcode/Android Studio.

### 🖥️ 5. Десктопная сборка (Windows / macOS via Electron)
- Конвертация Web приложения в `.exe` или `.app` файлы.
- Конфигурация electron-builder.

## 🛠️ Обновленные функции после доработки

Мы добавили мощные современные функции для комфортной разработки:

### 📥 1. Drag & Drop импорт файлов
- Перетаскивание файлов прямо в окно конструктора.
- Автоматическое чтение и импорт в дерево файлов.

### 🔐 2. Авторизация пользователей (User Auth)
- Вход/Регистрация.
- Профиль пользователя с сохранением истории.

### ☁️ 3. Сохранение в облако (Cloud Sync)
- Хранение проектов в Supabase или Firebase.
- Синхронизация истории изменений в один клик.

### 📦 4. Экспорт проекта в ZIP
- Скачивание всех сгенерированных и отредактированных файлов на компьютер как стандартный архив.

### 🎨 5. Полноценный менеджер тем с кастомными цветами
- Конструктор своих палитр цветов (CSS переменные).
- Переключение между пресетами (Dark Cyberpunk, Forest Mint и т.д.).

## 🗺️ План развития (Roadmap)

### 📊 Какие функции остались для реализации:
1. **Полноценная интеграция Git:** Синхронизация с GitHub прямо из дерева файлов.
2. **Мультиплеер-режим (Collaborative editing):** Совместная работа нескольких разработчиков одновременно.
3. **Развертывание на хостинг в 1 клик (Deploy):** Отправка на Vercel/Netlify.
4. **Магазин шаблонов (Templates Hub):** Готовые проекты для старта (e-commerce, блоги, CRM).

## 🚀 Инструкции по сборке

### Установка зависимостей:
```bash
npm install
```

### Разработка в Web:
```bash
npm run dev
```

### Сборка в EXE:
```bash
npm run build:exe
```

### Сборка в APK:
```bash
npx cap sync android
cd android && ./gradlew assembleDebug
```
